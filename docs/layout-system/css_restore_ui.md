# 还原UI设计
:::info 流程
1. 产品设计原型
2. UI设计页面
3. 前端还原UI
:::

:::tip 长度单位
- 绝对长度单位 自身就是长度
- 相对长度单位 rem vw
:::

:::caution 颜色分类
- 颜色关键字 英文单词
- RGB `rgb(255,255,255)` rgba
- HEX `#fffff` 16进制 `16*16=256 0-255`
- HSL `hsl(hue,saturation,lightless)` 色相/饱和度/亮度
:::

:::danger UI设计图 源文件类型
1. .psd Photoshop工具
2. .sketch Sketch工具
3. .xd Xd工具
:::
