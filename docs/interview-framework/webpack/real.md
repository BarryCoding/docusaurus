## 前端代码为何需要进行构建和打包
## module chunk bundle 区别
:::tip 答案
- module，chunk 和 bundle 是同一份逻辑代码在不同转换场景下的叫法。
- 项目的各个源码文件 是 module
- webpack 在处理这些代码时是 chunk
- 最后输出生为浏览器可运行的 bundle
:::
