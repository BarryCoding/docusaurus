为何 v-for 中要用key​

  • 必须使用key， 不推荐使用index/random
  • vdom比较时 确保vdom的唯一性 使用diff算法，通过tag类型和key来进行对比
  • 通过diff算法得出最少渲染内容 提升渲染性能
