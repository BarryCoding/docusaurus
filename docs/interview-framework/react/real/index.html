函数组件 类组件 区别​

  • 函数组件
    • 纯函数 输入props 返回jsx
    • 没实例 没生命(lifecycle) 没状态(state)
    • 不能扩展 没灵魂 只实现简单功能的工具人
  • 类组件
    • 有实力 有生命 有状态
    • 可扩展 有灵魂 能封装继承多态的大佬

什么是受控组件​

  • 表单值 受state直接控制
  • 需要监听 onChange事件 setState 更新状态

React 与 Vue 区别​

相同/类似
  • 组件化
  • 数据驱动视图
  • vdom 操作 DOM 提升性能

区别ReactVue

编程形式函数式 state不可变 用setState声明式 直接读取/修改data
